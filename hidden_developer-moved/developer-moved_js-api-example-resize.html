Here is an example of a basic setup of JW Player, with a link below that resizes it on the fly. This can be used by publishers to build a "lean back" video viewing experience into their site:

 

resize the player

This feature is built using the JW Player JavaScript API, with just a few lines of JavaScript code.

Embed Code

As always, ensure you have uploaded a copy of JW Player and included the script in the <head> of your page, together with your player key. See the Quick Start Guide for more info. Next, place the following embed code at the location you want the player to appear:

<div id="myElement"></div>

<script>
var playerInstance = jwplayer("myElement");
  playerInstance.setup({
    image: "/uploads/myPoster.jpg",
    file: "/uploads/myVideo.mp4",
    width: 320,
    height: 180
  });
</script>

Second, add the HTML and JavaScript to implement the resize link:

<a href="javascript:resizePlayer()">resize the player</a>

<script>
  function resizePlayer(){
    if(playerInstance.getWidth() == 640) { 
      playerInstance.resize(320,180); 
    } else {
      playerInstance.resize(640,360);
    }
  };
</script>

This script uses the playerInstance.resize() call to change the player dimensions on the fly. The actual resize button can be much prettified (e.g. to an iphone-style toggle, using 2 images). See the JavaScript API Reference for more info.

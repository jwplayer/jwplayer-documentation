Here is an example of a setup where the video controls are hidden. However, we are utilizing our own separate controls for playback. Each of these buttons is making calls with our API to control playback externally.

 



 

Embed code

Our example here is utilizing some simple API calls to reference our JW Player embed.
For example, the play button is using our method:

playerInstance.play();

Our skip forward/backward buttons are using two methods. First, we're getting our position in the video. Next, we're adding or subtracting 5 seconds from the position for our skip/rewind buttons.

​playerInstance.getPosition(); position -= 5; playerInstance.seek(position);​

and

​playerInstance.getPosition(); position += 5; playerInstance.seek(position);

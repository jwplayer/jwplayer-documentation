Looking for the API key and secret for your JW Player account? Here are the instructions to find them in your account dashboard:

  1. Log into your JW Player dashboard
  2. Click on the Account menu in the top right corner
  3. Choose Properties from the Account menu (only Admin users will be able to see Properties from this menu, see Managing Users for more details)



  4. Each property on your account will have a unique API key and secret, so you need to choose the property you want by clicking on the blue property name link or the gear button (see the green arrows below)



  5. The API key and secret are displayed on the property's details page. Do not share these details with anyone that you do not want to have direct access to all of your players and videos.
